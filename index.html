These feeds are a read-only, standards-compliant companion to Sri Lanka This Week, which is curated & maintained by Nuwan I. Senaratna (@nuuuwan) in the upstream repository nuuuwan/lk_news_digest.

The weekly digest is AI-generated from vetted English-language Sri Lankan news sources using large language models, as documented in the upstream project. This site simply parses the published markdown editions and exposes stable feed URLs for use with feed readers, applications, and integrations.
